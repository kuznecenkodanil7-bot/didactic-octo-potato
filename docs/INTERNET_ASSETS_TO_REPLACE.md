# Что искать в интернете для замены ассетов

Интернет-поиск в этой среде недоступен, поэтому URL не заполнены. Ниже — точные поисковые запросы и назначение каждого ассета.

## Рекомендуемые площадки для ручного поиска

- Poly Haven — CC0-текстуры и HDRI.
- ambientCG — CC0/PBR-текстуры.
- Kenney — бесплатные low-poly ассеты.
- Sketchfab — модели, проверять лицензию каждой модели.
- OpenGameArt — игровые ассеты, проверять лицензию.
- Quaternius — low-poly 3D packs.
- CGTrader / TurboSquid — чаще платные модели, проверять лицензию.
