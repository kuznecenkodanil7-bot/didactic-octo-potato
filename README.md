# Последнее письмо — 3D прототип

Это локальный браузерный WebGL-прототип психологической 3D-игры по концепции «Последнее письмо».

## Что уже сделано

- 3D-квартира: гостиная, коридор, спальня, кухня, кабинет, ванная.
- Управление от первого лица: WASD, мышь, E.
- Главная механика: старый почтовый ящик выдаёт письма Анны.
- 6 писем с выборами.
- Параметры: любовь, вина, безумие.
- Комнаты меняются по мере роста вины/безумия: трещины, пятна, силуэт, новые интерактивные объекты.
- 3 финала: вечная переписка, саморазрушение, отпускание.
- Все текстуры подписаны в `assets/asset_manifest.csv`, `assets/asset_manifest.json` и `docs/ASSET_MANIFEST.md`.
- В папке `assets/models` лежат OBJ-заглушки для всех моделей, чтобы было понятно, что заменять.

## Как запустить

Вариант 1 — быстро:

1. Откройте `index.html` в браузере Chrome/Edge/Firefox.
2. Нажмите «Начать».

Вариант 2 — правильнее для замены текстур:

1. Откройте терминал в папке проекта.
2. Запустите локальный сервер:

```bash
python -m http.server 8000
```

3. Откройте в браузере:

```text
http://localhost:8000
```

## Как заменять текстуры

1. Откройте `docs/ASSET_MANIFEST.md`.
2. Найдите нужную строку, например `TEX_mailbox_metal`.
3. Замените файл по пути `assets/textures/mailbox_metal.png` на свою картинку с таким же именем.
4. Перезапустите страницу.

Лучше использовать квадратные текстуры PNG/JPG 1024×1024 или 2048×2048.

## Как заменять модели

Сейчас игра рисует модели примитивами WebGL, а OBJ-файлы в `assets/models` — это подписанные заглушки под будущий перенос в Godot/Unity/Three.js.

Для полноценной версии лучше заменить эти слоты на модели формата `.glb/.gltf`:

- `mailbox_old` — старый почтовый ящик.
- `sofa_old` — старый диван.
- `bed_old` — кровать.
- `old_phone` — старый телефон.
- `glass_bottle` — бутылка.
- `anna_shadow` — силуэт Анны.

## Важно про интернет-ассеты

В этой среде интернет-поиск недоступен, поэтому реальные модели и текстуры из интернета не были скачаны. Вместо этого подготовлен полный список ассетов, их назначение и поисковые подсказки для замены.

Когда будут выбраны конкретные ассеты, нужно проверять лицензию: CC0, CC-BY, Royalty Free или разрешение на коммерческое использование.


## Публикация через GitHub Pages

Проект подготовлен для GitHub Pages. Можно загрузить всю папку в репозиторий и включить публикацию через GitHub Actions.

Главные файлы для GitHub:

- `.github/workflows/deploy-pages.yml` — автоматический деплой.
- `.nojekyll` — корректная отдача статических файлов.
- `GITHUB_PAGES_GUIDE.md` — пошаговая инструкция.
- `docs/GITHUB_UPLOAD_CHECKLIST.md` — чек-лист загрузки.

После публикации игра будет открываться по ссылке вида:

```text
https://ТВОЙ_НИК.github.io/last-letter-3d-game/
```
